# Portafolio de Proyectos Full-Stack

Este repositorio contiene una colección de **proyectos personales y académicos Full-Stack**, mostrando la evolución de mi aprendizaje y mi capacidad para desarrollar aplicaciones web completas con **frontend, backend y bases de datos**.

---

## 🌟 Contenido del repositorio

El repositorio incluye los siguientes proyectos:

- **Cafetería**: Aplicación en Python con gestión de inventario y control de productos usando programación orientada a objetos.  
- **Tienda Productos con Carrito**: Proyecto en JavaScript con lógica de carrito, consumo de API y manipulación dinámica del DOM.  
- **Banco**: Aplicación web de gestión financiera personal desarrollada en React + TypeScript, con integración a Supabase y despliegue en Vercel.  
- **Inmobiliaria**: Sistema en PHP + MySQL con arquitectura MVC para gestión de propiedades.  
- **Tienda (Angular)**: Proyecto de tienda online desarrollado con Angular, con estructura modular y componentes reutilizables.

---

## 🛠 Tecnologías utilizadas

- **Frontend**: React, Angular, JavaScript, Bootstrap  
- **Backend**: Node.js, Express, PHP, Python  
- **Bases de datos**: MySQL, Supabase  
- **Otras herramientas**: Vercel (deploy), Git, GitHub

---

## 🚀 Objetivo

Este repositorio tiene como objetivo **demostrar habilidades Full-Stack** y mostrando tanto la lógica de programación como la organización modular y buenas prácticas de desarrollo.
